# Frontend для XKCD Search Service

Веб-интерфейс для работы с микросервисами поиска комиксов XKCD.

## Функциональность

- **Авторизация**: вход в систему с токеном
- **Поиск комиксов**: обычный и индексный поиск по фразам
- **Статистика**: просмотр статистики базы данных и статуса обновлений
- **Администрирование**: запуск обновления БД и очистка данных
- **Мониторинг**: проверка статуса всех микросервисов

## Использование

### Локально

1. Откройте `index.html` в браузере
2. Убедитесь, что API сервис запущен на `http://localhost:28080`

### Через Docker Compose

Фронтенд автоматически доступен на `http://localhost:8080` после запуска:

```bash
make up
```

## API Endpoints

Фронтенд использует следующие endpoints:

- `POST /api/login` - авторизация
- `GET /api/ping` - проверка статуса сервисов
- `GET /api/search` - обычный поиск
- `GET /api/isearch` - индексный поиск
- `GET /api/db/stats` - статистика БД
- `GET /api/db/status` - статус обновления
- `POST /api/db/update` - запуск обновления (требует авторизацию)
- `DELETE /api/db` - очистка БД (требует авторизацию)
